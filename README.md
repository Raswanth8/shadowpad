# shadowpad
A npm package  for non-designers to add shadow and padding of images

# Installation

`npm i shadowpad --save`

```
import{ shadowpad } from 'shadowpad';

shadowpad({
        shadow_type: 'soft',
        padding: false
});

```
# Options available

* *shadow_type* - _hard/soft_
* *padding* - _boolean_
